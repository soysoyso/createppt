부동산 임대 PPT 생성기

⚠️ 새로고침 시 모든 데이터가 초기화됩니다
📋 사용 안내
① DB 엑셀 업로드
B_DB 시트(건물 기본정보) + 공실 시트(공실현황)를 자동 인식합니다.
빌딩코드로 두 시트가 자동 연동됩니다.
② 이미지 업로드
대표사진·도면을 각각 업로드 후, 건물 카드 하단 입력창에 확장자 없이 빌딩코드 파일명을 입력하세요.
입력창이 초록색이면 매칭 성공. 기본값은 빌딩코드 / 빌딩코드_plan으로 자동 입력됩니다.
③ 건물 선택
좌측 목록에서 원하는 건물만 체크하세요. 전체선택/해제 버튼도 제공됩니다.
④ PPT 생성
하단 "PPT 생성" 버튼 클릭 → 목록 슬라이드 + 건물별 슬라이드 자동 생성.
목록 슬라이드에서 건물명 클릭 시 해당 슬라이드로 이동합니다.
💡 성능 팁: 50개 이상 + 지도 포함 시 오래 걸립니다. PPT 설정에서 지도를 생략하면 빠르게 생성됩니다. 200개 이상은 100개씩 나눠서 생성을 권장합니다.
🗺 Kakao 지도 API 키
주소 정확도를 높이려면 Kakao REST API 키를 입력하세요.
developers.kakao.com → 앱 → 일반 → REST API 키
